# Graffiticode Training Examples

## Language L0165

### Prompt
"Create a 2 by 2 spreadsheet with the content being the names of the cells"

### Code

```
columns {
  A: {
    width: 50,
    justify: "center",
  },
  B: {
    width: 50,
    justify: "center",
  },
}
cells {
  A1: {
    text: "A1",
  },
  A2: {
    text: "A2",
  },
  B1: {
    text: "B1",
  },
  B2: {
    text: "B2",
  },
}
{}..
```
